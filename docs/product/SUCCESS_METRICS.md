
# Success Metrics (KPIs)

## 1. Business Metrics
*   **Payroll Accuracy**: 100% match with manual LHDN calculator.
*   **Time Saved**: Reduce payroll processing time from 3 days to < 1 hour.
*   **Attendance Integrity**: < 1% spoofed/fake clock-ins detected.

## 2. Technical SLOs (Service Level Objectives)
*   **Face Recognition Latency**: < 2 seconds for match.
*   **App Load Time**: < 1 second (LCP).
*   **Offline Reliability**: 100% of punches stored locally if offline, synced within 5s of reconnection.

## 3. Product Engagement
*   **Kiosk Usage**: Daily Active Users (DAU) / Total Staff.
*   **AI Adoption**: Number of AI queries per Admin per week.
